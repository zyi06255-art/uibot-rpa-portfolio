# 财务报表分析课程作业 / Financial Statement Analysis Coursework

## 项目简介

高校财务专业 RPA 课程大作业。使用 UiBot RPA 自动从财务报表 Excel 中提取数据，进行财务指标计算与分析，包括偿债能力、营运能力、盈利能力等维度的评估。

## 核心功能

### 1. 财务数据提取

从 Excel 财务报表中自动提取：
- 资产负债表（资产、负债、所有者权益）
- 利润表（收入、成本、利润）
- 现金流量表

### 2. 指标计算

| 维度 | 指标 |
|------|------|
| 偿债能力 | 流动比率、速动比率、资产负债率 |
| 营运能力 | 应收账款周转率、存货周转率、总资产周转率 |
| 盈利能力 | 销售净利率、总资产报酬率、净资产收益率 |

### 3. 格式转换

`格式转换.py` 负责将不同格式的财务数据统一标准化：
- 日期格式标准化
- 金额单位统一（元/万元/亿元）
- 科目名称映射

## 技术亮点

- 教育场景下的 RPA 应用展示
- 财务数据处理的 Python + UiBot 混合架构
- 动态公式计算（非硬编码）
- 知识库冷热分层架构的实际使用案例

## 项目背景

本项目为高校财务专业 RPA 课程的大作业，展示了如何将 RPA 技术应用于财务分析领域，实现报表数据的自动提取、计算和分析。
