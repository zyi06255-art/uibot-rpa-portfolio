# 发票查验自动化 / Invoice Verification Automation

## 项目简介

自动从发票图片/PDF 中提取关键信息（发票号码、开票日期、金额、税额、销售方等），与税务系统或数据库中的记录进行比对，完成发票真伪查验和信息核验。

## 核心流程

```
发票文件 (PDF/JPG/PNG)
  → OCR 文字提取
    → 关键字段结构化
      → 数据库比对
        → 查验结果输出
```

## 技术亮点

### 1. 结构化信息提取

从 OCR 输出的非结构化文字中提取结构化的发票字段：
- 发票代码、发票号码
- 开票日期
- 金额（不含税）、税额、价税合计
- 销售方名称、纳税人识别号
- 购买方名称、纳税人识别号

### 2. 多格式支持

- PDF 发票（数字版 + 扫描版）
- 图片发票（JPG, PNG）
- 电子发票 PDF

### 3. 数据校验

- 金额逻辑校验（不含税金额 + 税额 = 价税合计）
- 日期格式校验
- 发票号码格式校验
- 与数据库历史记录去重比对

## 文件说明

| 文件 | 角色 |
|------|------|
| `流程块.task` | 主流程：发票文件读取 |
| `流程块1.task` | OCR 识别调用 |
| `流程块2.task` | 数据校验与入库 |
| `generate_doc.py` | 查验报告生成（python-docx） |
| `uibot-api-full.md` | UiBot API 参考手册 |
| `res/invoices.json` | 发票结构化数据模板 |

## 运行说明

1. 将待查验发票放入 `res/` 目录
2. 在 UiBot 中运行主流程
3. 查验结果输出为 Word 报告（由 `generate_doc.py` 生成）
