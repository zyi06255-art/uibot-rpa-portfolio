# 自动导入收款单 / Auto Receipt Import

## 项目简介

端到端 RPA 流程：自动登录企业管理系统 → 读取收款单 Excel 数据 → 逐笔在系统中创建收款单据 → 记录处理状态。展示了完整的"登录→操作→注销"RPA 管线模式。

## 核心流程

```
系统登录 → 文件下载 → 数据读取 → 单据创建 → 状态反写 → 消息分发 → 异常处理
```

## 技术亮点

### 1. 知识库集成

本项目是**知识库冷热分层架构**的原型验证项目之一。项目目录中的 `knowledge-base/` 是知识库便携包的早期版本，记录了从 V1→V3 的演进过程。

### 2. 完整的 RPA 管线

- **系统登录**：处理登录页面的各种异常状态（已登录、密码错误、验证码等）
- **文件操作**：从共享目录或邮件附件下载收款单 Excel
- **数据录入**：逐行读取 Excel，在系统中创建收款单
- **状态追踪**：每笔单据的处理状态（成功/失败/跳过）记录到数据库

### 3. 容错设计

- 单笔单据创建失败不阻断整个流程
- 3 次重试 + 幂等检查的标准容错模式
- 详细的异常日志记录（精确到行号）
