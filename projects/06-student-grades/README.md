# 学生成绩管理 / Student Grade Management

## 项目简介

RPA 自动化处理学生成绩数据：从多个 Excel 成绩表中读取各科成绩，进行汇总、统计、排名计算，生成成绩分析报表。

## 核心流程

```
多个成绩Excel文件
  → 统一格式读取
    → 数据清洗与校验
      → 汇总统计（总分/平均分/排名）
        → 生成分析报表
```

## 技术要点

### 数据聚合

- 多 Sheet 动态读取（不硬编码 Sheet 名）
- 动态列映射（适配不同学校的成绩表格式）
- 成绩数据校验（异常分数自动标记）

### 统计分析

- 总分计算
- 平均分、最高分、最低分
- 排名（含并列处理）
- 各科成绩分布
